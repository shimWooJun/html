리눅스 기본 명령어 메뉴얼 작성


파일 시스템 탐색 기본 명령어


pwd

현재 작업 디렉토리를 표시한다.




cd

디렉토리를 이동할때 사용하는 명령어다.




ls

디렉토리에 있는 내용을 리스트로 출력한다.




file

지정된 파일의 타입을 확인하는 명령어다.




less

파일의 내용을 확인하는 명령어다.





파일과 디렉토리 조작 명령어


cp

파일, 디렉토리를 복사한다.




mv

디렉토리를 이동 시킬때 사용한다.




mkdir

디렉토리를 생성할때 사용한다.




rm

파일, 디렉토리를 삭제할때 사용한다.





ln

링크파일을 만드는 명령어다.





명령어를 다루는 명령어


type

명령 유형에 대한 정보를 표시하게 한다.




which

특정명령어의 위치를 찾아주는 명령어이다.




man

여러가지 명령어와 프로그램의 사용법을 확인할 수 있는 명령어다.





apropos

검색어와 관련있는 명령어를 설명과 함께 출력한다. man -k옵션과 같은 기능이다.




info

리눅스 명령어의 사용 방법, 옵션 등을 나타낸다.





whatis

명령어에 대한 기능을 간략하게 나타낸다.




alias

명령어 별칭을 확인하고 등록 해제할 수 있다.
